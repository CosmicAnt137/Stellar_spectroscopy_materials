### Project
**Physical Parameters of Stars in NGC 6397 Using Full-Spectrum Fitting**

The project involved exploring stellar atmospheric parameter estimation
(effective temperature, surface gravity, metallicity) using full-spectrum
fitting techniques applied to archival spectroscopic data of the globular
cluster NGC 6397.
